# jfut-scoop-bucket

The "jfut-scoop-bucket" bucket for [Scoop](http://scoop.sh).

To make it easy to install apps from this bucket, run

```
scoop bucket add jfut-scoop-bucket https://github.com/jfut/jfut-scoop-bucket.git
```
